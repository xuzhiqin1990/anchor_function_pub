# Anchor function: a type of benchmark function for studying language models

Main structure:
    
    ```
    ├── data_generator 
    ├── model
    ├── utils
    ├── main.py
    ├── data.py
    ├── train.py
    ├── script.py
    ```

Run python ``script.py`` to train the model.

Run ``paper_plot_code/ipynb`` to reproduce the results in the paper.
